START THE GAME

First, you will need to create the executable c file to run the program. Download the c file by clicking the "play" button. Be sure to navigate to the folder where the c file was downloaded to. Then, open the terminal and type:

gcc FizzBuzz.c -o FizzBuzz

This command creates an executable file called FizzBuzz. Using the terminal once again, type:

./FizzBuzz

The output of the program should now appear on your terminal.

OVERVIEW

FizzBuzz is a simple program that outputs all integers from 1 to 100. However, any integer that is divisible by 3 is replaced by "Fizz," any integer divisible by 5 is replaced by "Buzz," and any integer divisible by 3 and 5 is replaced by "FizzBuzz."
