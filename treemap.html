OVERVIEW

This implemenation of the TreeMap data structure uses a binary seach tree to store (key, value) pairs. This allows us to store and retreive (key, value) pairs in O(log N) time> Futhermore, we can retrieve the keys in sorted order.

The TreeMap uses resursive methods such as put, get, and remove to place keys into the TreeMap, get thier value, and remove keys. Furthermore, methods to return the minimum and maxiumum keys are utilized. Lastly, the method getKeys traverses the TreeMap in an in-order fashion to return the keys in sorted order.
