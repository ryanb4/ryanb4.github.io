DIJKSTRA'S ALGORITHM

  • Java

SOURCE

OVERVIEW

The first part of this project was to design and implement a directed graph data structure. This graph has directed and weighted edges between the verticies. This graph implementation also allows the user to provide each of the verticies with a label.

The second part of this project was to create an implementaion of Dijksra's shortest path algorithm by using the previously built digraph. This algorithm takes in the weighted digraph and a single starting vertex as input, and can produce information on the shortest path from the starting vertex to all of the other verticies.
